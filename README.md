# QuCS - Quantum Computer Systems Lecture Series

A website for the QuCS (Quantum Computer Systems) lecture series, built with Astro and Keystatic CMS.

## Features

- **Headless CMS**: Keystatic CMS with GitHub-based storage for content management
- **Multiple Editors**: GitHub OAuth authentication for team editing
- **Fast Search**: Pagefind-powered static search for lecture discovery
- **Lecture Management**: Lectures with speaker info, videos, slides, and categories
- **Responsive Design**: Mobile-first with dark mode support

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Deploy to Cloudflare Pages
npm run deploy
```

## Content Management

Visit `/keystatic` to access the CMS admin panel. Editors need write access to the GitHub repository.
